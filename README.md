# AeroSense-Air-Quality-Analysis
An end-to-end IT project using Machine Learning (Random Forest) , Power BI and MySQL for urban pollution forecasting.

## Overview
AeroSense is an analytical and predictive ecosystem designed to bridge the gap between historical climate patterns and future AQI forecasts.

## Features
* *Predictive Intelligence:* Random Forest Regressor achieving a R^2 of 0.99.
* *Interactive Dashboard:* Comprehensive Power BI visualization of seasonal AQI trends.
* *AI Chatbot:* Real-time query system developed in Google Colab.

## Tech Stack
* *Language:* Python (Scikit-Learn, Pandas, Numpy, Matplotlib)
* *Database:* SQL / CSV
* *Visualization:* Microsoft Power BI
* *Platform:* Google Colab
